Obq_SetAlpha


Class

Texture [Passthrough > Modify]


Output

Color RGBA


Synopsis

Obq_SetAlpha is a texture node that sets the alpha of the input color.


Usage

This node is quite self explanatory.




Color [color]

The color input.


Alpha [alpha]

The alpha you want to give the color parameter.
