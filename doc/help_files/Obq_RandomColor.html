Obq_RandomColor


Class

Texture [Generator]


Output

Color RGBA


Synopsis

Obq_RandomColor is a texture node that outputs a random color based on the shaded object's name.


Usage

Multiple objects can share a material with this node and it will output different colors.


Options


Number of values [randMax]

The maximum number of random colors.


Seed [seed]

An offset for the random number generator.


Strip


Model [stripModelName]

Strip the model name from the equation. If 2 objets has the same name but not the same model as parent, this will make both object have the same color.


Frame [stripFrameNumber]

Strip the frame number from the equation. If this isn't checked, the colors will change each frame.


Instances Frame [stripInstanceFrameNumber]

Special cases for instances.


Instances ID [stripInstanceID]

Special cases for instances. If this is not check, because all instances share the same pointcloud name, all instances will have the same color.


Instances Shape [stripInstanceFrameNumber]

Special cases for instances. All instances have a shape description in their Arnold name and this option strips it.


Inputs


Color # [color#]

The color inputs that are selected randomly based on the name of the shaded object.
